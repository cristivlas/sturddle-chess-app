When a command is not recognized, the app will fall back to the Assistant function, which uses two tiers: a local intent recognizer and integration with ChatGPT. When "Remote Assistant" is enabled in settings and commands are not understood, and the Local Assistant is turned off or fails to recognize text, the text is processed by the ChatGPT integration layer. For this to work, users need to provide their own OpenAI API Key and enter it into the settings.

The Assistant feature enables free-form commands, such as "What is the best move?", "Make a recommendation" or "I want to practice checkmate in three moves".

For more information on obtaining an API key, please visit the OpenAI API key page.
