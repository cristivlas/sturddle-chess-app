# sturddle-chess-app
Casual Chess App

- Original chess engine (C++ / Cython submodule)
- Built-in board editor
- Turn AI off / on
- Copy / paste game in PGN format (including comments)
- Voice interaction
- Optional opening book(s)
- Built-in puzzle collection
